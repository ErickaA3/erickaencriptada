FUNCION IF

Esta estructura lógica permite que la computadora tome una decisión según el resultado de una expresión. Un ejemplo de expresión es Variable1 > 0. Esta expresión es verdadera si el valor almacenado en la variable es mayor que cero. Cuando se encuentra una estructura IF, evalúa la expresión proporcionada. Si la expresión es falsa, la computadora continúa con la siguiente estructura, ignorando el contenido.

Else, el else va junto al if, si la condicion del if es falsa, se ejecuta el else.
